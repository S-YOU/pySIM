PC / SC

About

PC/SC is a specification for communication between Personal Computers and Smart Cards. Microsoft® has implemented these specifications for use in it's Windows operating systems.

Installation

Use the link below to obtain the installation program for Microsoft SmartCard Base Components.

Microsoft Win32 Smart Card Base Components

These smartcard base components are only for Microsoft Windows 95,98,ME & NT4. DO NOT install on win2000 or winXP. Support for PC/SC is native on Win2k and XP.

PC/SC Tools

PCSCInf.exe - Shows if Smartcard Base Components are installed.

PCSCDiag.exe - Provides information about the installed Smartcard Base Components and connected PC/SC card readers.

References

PC/SC workgroup - Provides the specifications for the PC/SC interfaces.
